<h2 align=center>Features</h2>

* [X] Run clients
* [X] CLI Interface
* [X] Download manager
* [X] Message system
* [X] Settings system
* [X] Config system
* [X] Cache system 
* [X] Packaging (CI/CD, PyInstaller)
* [X] Mirrors (4 servers)
